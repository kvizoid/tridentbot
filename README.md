# 🛒 Minecraft Shop Discord Bot

Discord бот для продажи игровых предметов на ванильном сервере Minecraft.

---

## 🚀 Установка

### 1. Требования
- Python 3.10+
- pip

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Создай `.env` файл
```
DISCORD_TOKEN=твой_токен_бота
```

### 4. Запусти бота
```bash
python bot.py
```

---

## ⚙️ Первоначальная настройка в Discord

1. **Создай нужные каналы** на сервере:
   - Канал магазина (например `#магазин`)
   - Канал оформления заказов (например `#оформить-заказ`)
   - Категорию для каналов заказов (например `📦 Заказы`)

2. **Включи Developer Mode** в настройках Discord (Настройки → Расширенные → Режим разработчика)

3. **Напиши `/setup`** в любом канале — откроется меню настроек

4. Через меню настроек задай:
   - ID канала магазина
   - ID канала оформления
   - ID категории заказов
   - Роль доставщиков
   - Роль администраторов

5. **Напиши `/refresh`** — бот отправит сообщения с интерфейсом в нужные каналы

---

## 📋 Команды

| Команда | Описание | Права |
|---------|----------|-------|
| `/setup` | Открыть меню настроек | Администратор |
| `/refresh` | Обновить сообщения в каналах | Администратор |
| `/orders` | Список активных заказов | Manage Messages |

---

## 🔄 Как работает заказ

```
Покупатель выбирает предметы в #магазин
         ↓
Переходит в #оформить-заказ → нажимает кнопку
         ↓
Заполняет форму: ник в MC + место доставки
         ↓
Создаётся канал в категории заказов + пинг доставщиков
         ↓
Доставщик нажимает "▶️ Начать сборку" → покупателю приходит уведомление
         ↓
После доставки нажимает "✅ Выполнен" → покупателю приходит уведомление
         ↓
Канал автоматически удаляется через 30 секунд
```

## 📦 Управление товарами

Через `/setup` → **Предметы** можно:
- Добавлять новые предметы
- Удалять предметы по ID

Или редактировать `items.json` вручную:
```json
[
  {
    "id": "уникальный_id",
    "name": "Название",
    "emoji": "🔹",
    "price": 100,
    "description": "Краткое описание"
  }
]
```

## 🔐 Права бота

Боту нужны следующие разрешения:
- `Send Messages`
- `Manage Channels`
- `Read Message History`
- `Use Application Commands`
- `Mention Roles`
- `Embed Links`
